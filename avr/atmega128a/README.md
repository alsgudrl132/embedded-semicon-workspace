# ATmega128 기반 간단 선풍기 제어 시스템

## 📌 프로젝트 개요
ATmega128 마이크로컨트롤러를 이용해 선풍기의 속도를 제어하고, 속도에 따라 시각적으로 표시되는 시스템을 구현한 프로젝트입니다.

## ⚙ 사용 부품 및 환경
- ATmega128
- Sub Motor & Fan
- 스위치
- LED (속도 표시용)
- FND Segment (속도 표시용)

## 🔧 주요 기능
- 버튼을 눌러 팬 속도 조절  
  → 정지 → 중속 → 고속  
- 속도에 따라 LED 점등 패턴 또는 LCD 표시 변화  
- PWM 신호를 통해 팬 속도 제어
